# HR Analytics Dashboard (Power BI)

## Overview
This project is an HR analytics dashboard built in Power BI to analyze employee attrition and workforce trends across departments, salary groups, experience levels, and job roles.

## Dashboard Highlights
- Total Employees: 1K
- Employees Left: 237
- Attrition Rate: 16.12%
- Average Salary: ₹6.50K

## Key Insights
- Attrition by job role
- Attrition by gender
- Attrition by department
- Attrition by salary group
- Attrition by experience level
- Employee filtering by department, gender, and education level

## Tools Used
- Power BI
- Excel / CSV
- Kaggle Dataset
